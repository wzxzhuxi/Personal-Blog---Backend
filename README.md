## 快速开始

1. 克隆仓库到本地：
   ```bash
   git clone https://github.comw/zxzhuxi/Personal-Blog---Backend.git
2. 进入项目目录：
   ```bash
   cd Personal-Blog---Backend-master
4. 安装依赖：
   ```bash
   ./mvnw install
6. 运行项目：
   ```bash
   ./mvnw spring-boot:run
